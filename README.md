## 🚀 Sobre mim

Analista de Dados na **3CORP Technology**, com foco total em **Power BI, SQL e automação de processos com Python**.

Transformo dados brutos em **dashboards executivos**, **modelos de dados** e **pipelines** que geram decisão real para o negócio.

Atualmente evoluindo em:

- 📊 **Power BI** — dashboards, DAX, modelagem
- 🗄️ **SQL** — consultas, modelagem relacional, performance
- 🐍 **Python** — automação e ETL
- 🐘 **PostgreSQL**, **MySQL** e **SQLite**
- ⚙️ **Engenharia de Dados** — pipelines & integrações
- ✅ Boas práticas e governança de dados
